## 4. Eksplorasi Data

Berikut adalah beberapa analisis eksplorasi data yang dapat dilakukan:

### 4.1 Distribusi Penjualan Handphone

Distribusi penjualan handphone memberikan gambaran umum tentang jumlah penjualan handphone di Tokopedia pada kuartal kedua 2024.

```python
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

# Membaca data dari file Excel
df = pd.read_excel('Handphone_Clear.xlsx')

# Menampilkan beberapa baris pertama data
print(df.head())

# Menggambarkan distribusi jumlah terjual
plt.figure(figsize=(10, 6))
sns.histplot(df['jumlah_terjual'], bins=30, kde=True)
plt.title('Distribusi Penjualan Handphone di Tokopedia (Kuartal 2 2024)')
plt.xlabel('Jumlah Terjual')
plt.ylabel('Frekuensi')
plt.show()
```

### 4.2 Tren Penjualan Berdasarkan Harga

Analisis ini menunjukkan hubungan antara harga dan jumlah penjualan.

```python
plt.figure(figsize=(10, 6))
sns.scatterplot(data=df, x='harga', y='jumlah_terjual')
plt.title('Tren Penjualan Berdasarkan Harga')
plt.xlabel('Harga')
plt.ylabel('Jumlah Terjual')
plt.show()
```

### 4.3 Penjualan Berdasarkan Wilayah

Mengetahui wilayah mana yang memiliki penjualan handphone tertinggi dapat memberikan wawasan tentang pasar potensial.

```python
plt.figure(figsize=(12, 8))
sns.countplot(data=df, y='wilayah', order=df['wilayah'].value_counts().index)
plt.title('Penjualan Handphone Berdasarkan Wilayah')
plt.xlabel('Jumlah Terjual')
plt.ylabel('Wilayah')
plt.show()
```

### 4.4 Rating Produk dan Jumlah Penjualan

Analisis ini menunjukkan hubungan antara rating produk dan jumlah penjualan.

```python
plt.figure(figsize=(10, 6))
sns.boxplot(data=df, x='rating', y='jumlah_terjual')
plt.title('Jumlah Penjualan Berdasarkan Rating Produk')
plt.xlabel('Rating')
plt.ylabel('Jumlah Terjual')
plt.show()
```

### 4.5 Produk Terlaris

Mengetahui produk-produk terlaris dapat memberikan gambaran tentang produk yang paling diminati oleh konsumen.

```python
top_products = df.groupby('nama_produk')['jumlah_terjual'].sum().nlargest(10)

plt.figure(figsize=(12, 8))
sns.barplot(x=top_products.values, y=top_products.index)
plt.title('10 Produk Handphone Terlaris di Tokopedia (Kuartal 2 2024)')
plt.xlabel('Jumlah Terjual')
plt.ylabel('Nama Produk')
plt.show()
```
